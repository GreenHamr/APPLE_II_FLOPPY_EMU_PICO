# Инсталация на FatFS библиотека

## Важно

По подразбиране проектът използва локална опростена версия на FatFS (`ff.c` от корена на проекта).
За пълна функционалност, препоръчително е да добавите официалната FatFS библиотека.

## Метод 1: Изтегляне от официалния сайт (препоръчително)

1. Посетете: http://elm-chan.org/fsw/ff/00index_e.html
2. Изтеглете последната версия (R0.15 или по-нова)
3. Разархивирайте архива
4. Копирайте файловете от `00fatfs/` директорията в `fatfs/`:
   ```bash
   cd /home/CurrentProjects/PRAVETZ-8D/PROJECT_P8D_V5.2/FIRMWARE/Floppy_PICO/Floppy_PICO_green
   # След като изтеглите и разархивирате:
   cp <изтеглена_директория>/00fatfs/ff.c fatfs/
   cp <изтеглена_директория>/00fatfs/ff.h fatfs/
   # ffconf.h може да се копира от корена на проекта или да се използва версията от архива
   ```

## Метод 2: Използване на git (ако имате достъп)

```bash
cd /home/CurrentProjects/PRAVETZ-8D/PROJECT_P8D_V5.2/FIRMWARE/Floppy_PICO/Floppy_PICO_green
git clone https://github.com/elmchan/ff.git temp_fatfs
cp temp_fatfs/00fatfs/ff.c fatfs/
cp temp_fatfs/00fatfs/ff.h fatfs/
rm -rf temp_fatfs
```

## Проверка

След инсталацията, в `fatfs/` директорията трябва да има:
- `ff.c` - основният файл на FatFS (~50KB)
- `ff.h` - заглавният файл

## Конфигурация

Файлът `ffconf.h` се използва от корена на проекта. Ако искате да използвате версията от официалната библиотека, копирайте я също в `fatfs/`.

## Забележка

Ако не добавите файловете в `fatfs/`, проектът автоматично ще използва локалната опростена версия (`ff.c` от корена на проекта), която има ограничена функционалност.

# FatFS Library

Тази директория трябва да съдържа официалната FatFS библиотека.

## Инсталация

За да добавите официалната FatFS библиотека:

1. Изтеглете последната версия от: http://elm-chan.org/fsw/ff/00index_e.html
2. Или използвайте git:
   ```bash
   git clone https://github.com/elmchan/ff.git temp
   cp -r temp/00fatfs/* .
   rm -rf temp
   ```

3. Уверете се, че следните файлове са налични:
   - `ff.c` - основният файл на FatFS
   - `ff.h` - заглавният файл
   - `ffconf.h` - конфигурационен файл (може да се копира от корена на проекта)

## Алтернатива

Ако не добавите файловете тук, проектът ще използва опростената версия от корена на проекта (`ff.c`).
